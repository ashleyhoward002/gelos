## ğŸ’° Feature: Expenses

**Purpose:** Track spending, split costs fairly, manage group money

### Expense Entry:
| Field | Description |
|-------|-------------|
| Title | "Dinner at Giovanni's" |
| Amount | Total cost |
| Paid by | Which member paid |
| Date | When it happened |
| Category | Food, activities, transport, lodging, other |
| Attach to outing | Optional â€” link to specific outing |
| Receipt photo | Optional â€” snap the receipt |

### Split Modes:

**1. Equal Split**
- Total Ã· number of people
- Select who's included (members + guests)
- Example: $120 Ã· 4 = $30 each

**2. Custom Amounts**
- Manually enter what each person owes
- Example: Sarah $18, Mike $42, etc.
- System validates amounts add up to total

**3. Percentage Split**
- Assign percentage to each person
- Example: 3 people at 33.3% each
- Good for: couples splitting as a unit

### Guests (Non-Members):
- Add temporary people: "Sarah's boyfriend" or "Maria's cousin"
- They're included in splits for that expense
- Don't have app accounts â€” member responsible for collecting from them
- Tracked under the member who added them

### Balances:
- Personal dashboard showing:
  - What you owe others
  - What others owe you
  - Net balance
- Simplified debts: "You owe Sarah $45" (not itemized chaos)

### Private Reminders:
- Nudge someone who owes you
- Only visible to sender and recipient
- Not posted to group
- Friendly templates: "Hey! Just a reminder about the $30 from dinner ğŸ˜Š"

### Mark as Settled:
- When paid back, mark the debt as settled
- Clears from balance
- History preserved
